Implementation Strategy

1. Data Loading:

  • ✓ Guests already loaded from Firebase with householdId
  • ✓ Group guests by householdId in memory

2. Guest Selection Flow:

  • When guest selected → fetch household members from cached data
  • Show modal: "Seat together?" with household member list
  • Track household selection (Yes/No)

3. Guest Addition:

  • If "Seat Together": add all household members
  • If "Just this guest": add only selected guest
  • Check if any household members seated elsewhere

4. Household Reassignment:

  • If household member found in different table
  • Remove them from old table
  • Add them to new table with rest of household
